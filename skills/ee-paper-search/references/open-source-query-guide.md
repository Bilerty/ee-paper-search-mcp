# Open Source Query Guide

Open-source APIs usually work better with natural keyword queries than Scopus syntax.

## Query Style

OpenAlex:

```text
power system restoration distribution network reconfiguration microgrid resilience
```

Crossref:

```text
"power system restoration" "distribution network"
```

Semantic Scholar:

```text
microgrid fault management resilience optimization
```

Unpaywall:

```text
Use DOI values from candidate papers. Do not run keyword search.
```

## Use

- Use OpenAlex to catch records missed by Scopus and collect open metadata.
- Use Crossref to normalize DOI and publication metadata.
- Use Semantic Scholar to supplement citations, abstracts, and fields of study.
- Use Unpaywall only for OA status and OA links after DOI is known.
